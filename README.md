---

## 📌 Description

This is a simple Python project for generating secure random passwords using user preferences.

The program allows you to:

* Choose password length (minimum 8 characters)
* Select what characters to include:

  * Letters (uppercase/lowercase)
  * Numbers
  * Symbols
  * Or combinations of them
* Generate a secure password using the `secrets` module
* Optionally save the password to a `.txt` file on your Desktop

## 🚀 Features

## 📌 Описание

Это простой проект на Python для генерации безопасных случайных паролей с учетом предпочтений пользователя.

Программа позволяет:

* Выбрать длину пароля (минимум 8 символов)
* Выбрать состав пароля:

  * Буквы (верхний/нижний регистр)
  * Цифры
  * Символы
  * Или их комбинации
* Сгенерировать безопасный пароль с помощью модуля `secrets`
* При желании сохранить пароль в `.txt` файл на рабочем столе
